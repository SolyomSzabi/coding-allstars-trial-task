एक्सएचटीएमएल नामस्थान

नाम स्थान नाम http://www.w3.org/1999/xhtml उपयोग के लिए अभिप्रेत है विभिन्न विशिष्टताओं में जैसे:

सिफारिशें:

  • HTML 5: A vocabulary and associated APIs for HTML and XHTML
  • XHTML™ 1.0: The Extensible HyperText Markup Language
  • XHTML Modularization
  • XHTML 1.1
  • XHTML Basic
  • XHTML Print
  • XHTML+RDFa

अन्य दस्तावेज:

  • HTML 5.1
  • HTML 5.2

निम्नलिखित डब्ल्यू 3 सी कार्य समूहों के चार्टर्स में एचटीएमएल पर काम शामिल है इस नामस्थान को प्रभावित कर सकता है:

  • Web Platform Working Group, chartered October 2015

एक्सएमएल नेमस्पेस के बारे में अधिक जानकारी के लिए, कृपया देखें में नामस्थान एक्सएमएल।
